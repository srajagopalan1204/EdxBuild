SOP Modules by Entity

Select an entity to see only the SOPs that apply to that part of the business. Under each entity, modules are grouped by function such as Sales, Operations, Service, and Specialty.

Palco Power Systems

Service

  • TechMobile – Mobile Application for Technicians

  • Warranty – Warranty Service Processing
  • StartUp – Generator Startup Service

Core Business

Sales and parts processes that support generator projects.

Electrical Distribution – Branch Operations

Use these modules if you work in branch inside sales, counter sales, warehouse operations, or purchasing for Electrical Distribution.

Sales (SLS)

Operations (OPS)

Picking, shipping, and invoicing stock and direct orders.

  • Pick2Inv – Picking to Invoicing

Purchasing

  • Rrar – Replenishment Recommendation Report (Coming soon)

Buncher

  • CDL – Consolidated Billing Blanket Order Create (Coming soon)
